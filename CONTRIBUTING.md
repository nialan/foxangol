# Contributing

## Workflow

1. Create a feature branch.
2. Make changes.
3. Review changes with git diff.
4. Commit using an English commit message.
5. Push the branch.
6. Open a Pull Request.
7. Review the Netlify preview.
8. Merge after approval.

## Images

Store images in:

docs/blog/posts/images/

Use descriptive filenames.

## Markdown

Use Markdown whenever possible.

Use HTML only for responsive images.
